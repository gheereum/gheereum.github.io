What is antivirus?


(Source: https://www.helpsystems.com/blog/endpoint-antivirus-software-what-it-and-why-you-need-it)


Antivirus is not only a program developed to locate and delete viruses that have infected a device, but also a program that can prevent virus infections entering the computer system in the first place.

The program regularly scans all the files and directories on the computer system to spot any malware threats. Most programs also monitor network traffic to and from the computer system and is built to recognise suspicious software communicating over an unauthorised port. Consequently, the program will alert the user of the system to the activity.

Some antivirus software packages come with a browser extension which provides a warning message when visiting a potentially dangerous website. They may also alert the user of any privacy issues such as the existence of tracking cookies within the site.

Advantages of Using Antivirus Software


(Source: "https://www.israelglobalgateway.com/2017/10/iot-spyware-and-privacy-risks/")


With the use of computer systems increasing globally, the need for privacy and online security increases with it.

Antivirus Software provides:

  • Protection from Viruses & Spyware

    • Detects threats such as viruses, spyware, malware and removes them before they cause any damage to the data held on the system.

  • Protection from Phishing Attacks

    • Identifies any unauthorised attempts by a third party to access data within the system and takes necessary actions to alert the user and prevent any attacks.

  • Scan of removable devices

    • Provides the option to carry out a quick scan on removable devices such as external drives to detect any possible threats.

  • Two-Way Firewall

    • Carries out relevant checks on both incoming and outgoing data through the internet and blocks any suspicious data identified during the transmission.
